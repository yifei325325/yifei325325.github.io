表格：

注意第二行的“：”，每个单元格中，如果冒号靠左，就表示内容左对齐，冒号靠右，就表示内容右对齐，如果两边都有，就表示内容居中对齐

无序列表：

使用“-” 加一个空格，后面跟随内容，如下：

  • Python
  • Java
  • C
  • C++

有序列表：

使用数字加一个英文句号，加空格，再跟随内容，如下：

  1. Python
  2. Java
  3. C
  4. C++
